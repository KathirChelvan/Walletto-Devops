
# Walletto - Personal Finance Management Website with AI teacher

## Overview
Walletto is a comprehensive personal finance management application that helps users track their expenses, manage investments, and achieve their financial goals. Built with React, Firebase, and Tailwind CSS, it offers a secure and user-friendly interface for managing personal finances.

## Features

### 1. User Authentication
- Secure email and password authentication
- Protected routes and user sessions
- Password reset functionality
- SSL-secured data transmission

### 2. Financial Management
- Income tracking (Monthly salary and passive income)
- Expense categorization
  - Recurring expenses
  - Variable expenses
  - One-time expenses
- Bank balance monitoring
- Emergency fund tracking
- Investment portfolio management

### 3. Personal Finance Tools
- Financial goal setting and tracking
- Emergency fund status monitoring
- Investment type management
- Debt tracking
  - Loans
  - Credit card dues
  - EMI details

### 4. Asset & Liability Management
- Bank account management
- Property ownership tracking
- Vehicle ownership tracking
- Comprehensive financial overview

## Technical Stack

### Frontend
- React.js
- Tailwind CSS for styling
- React Router for navigation

### Backend
- Firebase Authentication
- Firebase Firestore
- Real-time data synchronization

### Security
- 256-bit SSL encryption
- Secure data storage
- Protected API endpoints

## Installation

1. Clone the repository:
```bash
git clone : https://github.com/suriya-shivaprakash/walletto

cd walletto
```

2. Install dependencies:
```bash
npm install
```

3. Configure Firebase:
   - Create a Firebase project
   - Update the Firebase configuration in `index.html`
   - Enable Authentication and Firestore

4. Start the development server:
```bash
npm start
```
